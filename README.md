## Usage

_NOTE: Requires [GNU Stow](https://www.gnu.org/software/stow/) to automatically symlink config packages._

```bash
# view help text
./dotfiles.sh -h

# dry run with default packages
./dotfiles.sh

# install default packages
./dotfiles.sh -i
```
